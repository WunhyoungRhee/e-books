30초면 끝나요, 진짜로

앱스토어의 터미널 버전

앱스토어에서 앱 설치해본 적 있죠? 검색하고, "설치" 버튼 누르고, 끝. 그거랑 비슷한데, 터미널 버전이에요.

버튼 대신 명령어를 타이핑하는 것뿐이에요. 진짜 30초면 끝나요. (준비물 확인 시간 빼고요!)

Step 0: 준비물 확인

설치하기 전에, 필요한 도구들이 다 있는지 먼저 확인해볼게요. 터미널을 열고 아래 명령어를 하나씩 쳐보세요.

Git 확인

터미널에 이렇게 타이핑하고 엔터:

git --version

성공: git version 2.43.0 같은 버전 번호가 나오면 OK!

실패: "command not found"가 나오면, git-scm.com에서 Git을 먼저 설치하세요.

Node.js 확인

node --version

성공: v20.11.0 같은 버전 번호가 나오면 OK! (v18 이상이면 돼요)

실패: nodejs.org에서 LTS 버전을 설치하세요.

Claude Code 확인

claude --version

성공: 버전 번호가 나오면 OK!

실패: Claude Code를 먼저 설치해야 해요. Anthropic 공식 문서를 참고하세요.

전부 OK인가요?

세 개 다 버전 번호가 나왔으면, 축하해요! 바로 Step 1로 넘어가세요. 하나라도 안 되면 해당 도구를 먼저 설치하고 오세요. 설치는 전부 "다운로드 → 다음 → 다음 → 완료" 형태예요.

Step 1: gstack 설치 명령어

자, 이제 진짜 설치합니다. Claude Code를 실행하고, 아래 명령어를 붙여넣기 하세요:

/install-skills https://github.com/garrytan/gstack

이게 끝이에요. 진짜로.

Claude Code가 gstack 저장소에서 28개의 스킬 파일을 다운로드하고 자동으로 설치해줍니다. 진행 상황이 터미널에 표시되니까 지켜보세요.

쉬운 비유

앱스토어에서 "gstack" 앱을 검색하고 "설치" 버튼을 누른 거예요.
다운로드 → 설치 → 완료! 알아서 다 해줍니다.

Step 2: CLAUDE.md에 등록하기

설치가 끝나면, gstack을 CLAUDE.md 파일에 등록해야 해요.

CLAUDE.md가 뭔데요?

CLAUDE.md는 Claude Code의 "설정 파일"이에요. "나는 이런 스킬들을 사용할 거야, 이런 규칙을 따라줘"라고 Claude Code에게 알려주는 파일이에요.

스마트폰의 "설정" 앱이라고 생각하면 돼요. 배경화면, 알림 설정, 언어 설정... 그런 걸 CLAUDE.md에서 하는 거예요.

프로젝트 폴더의 CLAUDE.md 파일에 gstack 스킬 경로를 추가하면 됩니다. 보통 설치 과정에서 자동으로 추가되지만, 확인해보세요.

CLAUDE.md 파일을 열면 이런 내용이 있을 거예요:

# Skills
- /path/to/gstack/skills/*.md

이 줄이 있으면, Claude Code가 gstack의 모든 스킬을 인식한다는 뜻이에요.

Step 3: 설치 확인 — /office-hours로 테스트!

설치가 잘 됐는지 확인하는 가장 좋은 방법은 직접 써보는 거예요.

Claude Code에서 이렇게 타이핑해보세요:

/office-hours

AI가 "어떤 아이디어나 프로젝트에 대해 이야기하고 싶으신가요?" 같은 질문을 하면 — 성공! gstack이 제대로 설치된 거예요.

"해당 스킬을 찾을 수 없습니다" 같은 에러가 나오면, CLAUDE.md 파일을 다시 확인해보세요.

Windows 사용자 주의사항

Windows에서는 가끔 경로(path) 문제가 생길 수 있어요. 특히 한글 사용자명이 포함된 경로에서요.

만약 설치 중 에러가 발생하면:

  • 프로젝트 폴더를 C:\projects 같은 짧고 영문인 경로에 만들어보세요.
  • 관리자 권한으로 터미널을 열어보세요 (터미널 아이콘 → 우클릭 → "관리자 권한으로 실행").
  • 그래도 안 되면, 챕터 13(문제 해결)을 참고하세요.

(선택) 팀원과 공유하기

혼자 쓰는 것도 좋지만, 팀이 있다면 같이 쓰면 더 좋아요.

gstack은 프로젝트의 CLAUDE.md 파일에 등록되기 때문에, 이 파일을 Git으로 공유하면 팀원들도 바로 같은 스킬을 사용할 수 있어요.

쉬운 비유

레시피북 한 권을 사서 팀 공유 책상에 올려놓은 거예요.
누구든 와서 레시피를 보고 따라 할 수 있어요.
개인별로 따로 살 필요 없이, 한 번 설정하면 팀 전체가 사용할 수 있습니다.

잠깐, 정리!

  • Step 0: Git, Node.js, Claude Code가 설치되어 있는지 확인
  • Step 1: Claude Code에서 /install-skills 명령어로 gstack 설치 (30초!)
  • Step 2: CLAUDE.md 파일에 gstack이 등록되어 있는지 확인
  • Step 3: /office-hours로 설치 확인
  • Windows 사용자는 영문 경로를 추천해요.
  • CLAUDE.md를 Git으로 공유하면 팀원도 바로 사용 가능!
