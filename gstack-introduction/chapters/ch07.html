CHAPTER 7

핵심 스킬 28개 한눈에 보기

전부 외울 필요 없어요. 5개만 기억하세요

스킬 카탈로그 보는 법

28개 스킬을 전부 외우려고 하지 마세요. 처음에는 5개만 알면 충분해요. 나머지는 "아, 이런 것도 있구나" 정도로 훑어보고, 필요할 때 찾아서 쓰면 됩니다.

각 스킬 옆에 난이도 배지가 있어요:

  • 초급 — 코딩 지식 없이도 바로 사용 가능
  • 중급 — 기본 개념을 알면 더 잘 활용 가능
  • 고급 — 개발 경험이 있으면 좋음

초급 배지가 붙은 스킬부터 시작하세요!

워크플로우 스킬 (20개)

프로젝트를 처음부터 끝까지 진행하는 데 필요한 스킬들이에요.

스킬 난이도 하는 일
/office-hours 초급 아이디어를 6가지 질문으로 검증하고 정리해요.
/conductor 초급 프로젝트 전체 계획서(PLAN.md)를 만들어요.
/plan-ceo-review 초급 CEO/창업자 관점에서 계획을 검토해요.
/plan-design-review 중급 디자이너 관점에서 UX를 검토해요.
/plan-eng-review 고급 엔지니어링 관점에서 기술 구현을 검토해요.
/design-consultation 중급 색상, 글꼴, 레이아웃 등 디자인 시스템을 만들어요.
/design-review 중급 완성된 결과물의 시각적 품질을 검토하고 수정해요.
/review 초급 코드 품질, 보안, 테스트 등 전반적인 품질을 검토해요.
/qa 중급 웹사이트를 브라우저로 열어서 버그를 찾고 수정해요.
/qa-only 중급 버그를 찾아서 보고만 해요 (수정은 안 함).
/ship 중급 코드를 정리하고, PR(Pull Request)을 만들어서 배포해요.
/land-and-deploy 고급 PR을 머지하고, 배포하고, 프로덕션 상태를 확인해요.
/retro 초급 스프린트를 되돌아보고 개선점을 정리해요.
/cso 고급 보안 감사를 수행해요 (OWASP Top 10, 위협 모델링 등).
/investigate 중급 버그의 근본 원인을 찾아내요.
/simplify 중급 변경된 코드의 재사용성과 효율성을 점검해요.
/document-release 중급 배포 후 문서를 자동으로 업데이트해요.
/setup-deploy 고급 배포 환경 설정 (Vercel, Fly.io 등)을 도와줘요.
/canary 고급 배포 후 실시간 모니터링을 해요.
/benchmark 고급 성능 측정 및 비교를 해요.

파워 툴 (8개)

특정 상황에서 사용하는 강력한 보조 도구들이에요.

스킬 난이도 하는 일
/browse 중급 헤드리스 브라우저로 웹사이트를 열고, 스크린샷 찍고, 상호작용해요.
/codex 고급 OpenAI Codex CLI를 활용한 독립 코드 리뷰 및 도전 모드.
/careful 초급 위험한 명령어 실행 전 경고해줘요. 초보자 필수!
/guard 초급 위험 명령어 경고 + 특정 폴더만 수정 가능하게 제한해요.
/freeze 중급 특정 폴더만 수정 가능하도록 잠가요.
/unfreeze 중급 freeze로 잠근 것을 풀어요.
/autoplan 고급 CEO, 디자인, 엔지니어링 리뷰를 자동으로 순차 실행해요.
/setup-browser-cookies 고급 실제 브라우저 쿠키를 가져와서 로그인된 상태로 테스트해요.

초보자가 먼저 배울 5개 스킬

28개가 많아 보이지만, 처음에는 이 5개만 기억하세요:

  1. /office-hours — 아이디어 정리의 시작
  2. /conductor — 계획서 만들기
  3. /plan-ceo-review — 계획 검토하기
  4. /review — 품질 점검하기
  5. /careful — 안전하게 작업하기

쉬운 비유

운전면허 딸 때도 처음부터 고속도로를 달리진 않잖아요?
먼저 직진, 좌회전, 우회전, 주차, 후진 — 이 5가지를 배우고,
나머지는 운전하면서 자연스럽게 익히는 거예요.

gstack도 마찬가지! 5개로 시작하고, 나머지는 필요할 때 하나씩 추가하세요.

잠깐, 정리!

  • gstack에는 워크플로우 스킬 20개와 파워 툴 8개, 총 28개의 스킬이 있어요.
  • 각 스킬에는 난이도 배지가 있어요: 초급, 중급, 고급.
  • 초보자는 5개 스킬만 먼저 익히세요: /office-hours, /conductor, /plan-ceo-review, /review, /careful.
  • 나머지는 필요할 때 하나씩 배우면 충분해요.
