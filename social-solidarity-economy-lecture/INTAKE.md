_conductor artifact — Phase 1 완료 (2026-04-13)_

## 인테이크 요약

| 항목 | 결정 사항 |
|---|---|
| **목표** | 사회연대경제 입문 특강 자료 일체 제작 (중학생도 이해 가능한 수준, 일반 성인도 활용 가능) |
| **형식/매체** | 강의 계획서, 전체 스크립트, 슬라이드 실제 파일(이미지 포함), 퀴즈형 활동지, 용어 정리표 |
| **플랫폼** | 로컬 마크다운 + Canva 슬라이드 |
| **대상** | 중학생 이상 일반 입문자 / 강의자: 일반 강사 |
| **핵심 톤** | 시대적 절실함이 피부로 느껴지도록 — 공감과 감정 이입을 통한 필요성 전달 |
| **언어 규칙** | 한국어 기본 / 전문·국제 용어는 영문 병기 / 어려운 용어는 주석 |
| **제약** | 정치적 중립, 한국 사례 우선, 대화체 친근한 톤 |
| **품질** | 추상 설명 배제 → 구체 사례·감성적 스토리텔링 우선 |

**프로젝트 유형:** 콘텐츠 (교육 자료)
