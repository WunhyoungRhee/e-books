참고 자료 및 링크

더 배우고 싶을 때, 여기서 시작하세요

이 부록에 대하여

이 책을 다 읽은 후에도 계속 배우고 싶을 때 활용할 수 있는 자료를 모았어요. 카테고리별로 정리했으니, 필요한 것을 골라서 살펴보세요.

참고: 링크는 작성 시점 기준이며, 시간이 지나면 변경될 수 있어요.

Claude Code 관련

  • Claude Code 공식 문서
    Claude Code의 설치, 설정, 사용법에 대한 공식 가이드. 가장 정확하고 최신 정보.
  • Claude Code GitHub 저장소
    Claude Code의 소스 코드와 이슈 트래커. 버그 리포트나 기능 제안은 여기서.
  • Anthropic 공식 사이트
    Claude AI를 만든 Anthropic의 공식 사이트. 최신 소식과 업데이트.
  • 스킬 패키지 공식 문서
    스킬 패키지의 구조, 프론트매터, references 등에 대한 공식 레퍼런스.

스킬 패키지 예시

  • 공식 스킬 예시 모음
    Anthropic에서 제공하는 공식 스킬 패키지 예시들. 구조를 배우기에 최고.
  • 커뮤니티 스킬 저장소들
    GitHub에서 "claude-code-skill" 또는 "claude-skill-package"로 검색하면 다양한 커뮤니티 스킬을 찾을 수 있어요.

프롬프트 엔지니어링

  • Anthropic 프롬프트 엔지니어링 가이드
    Claude에 최적화된 프롬프트 작성법. 공식 가이드라 신뢰할 수 있어요.
  • Prompt Engineering Guide (한국어)
    프롬프트 엔지니어링의 기초부터 고급 기법까지. 한국어 번역 제공.
  • Learn Prompting
    프롬프트 엔지니어링 무료 온라인 강좌. 체계적으로 배우고 싶을 때.

Git과 GitHub

  • Git 공식 사이트
    Git 다운로드와 공식 문서. 설치가 필요하면 여기서.
  • GitHub 공식 문서 (한국어)
    GitHub의 모든 기능에 대한 공식 가이드. 한국어로 읽을 수 있어요.
  • Learn Git Branching (한국어)
    Git의 브랜치 개념을 시각적으로 배울 수 있는 인터랙티브 사이트. 강력 추천!
  • YouTube "Git GitHub 초보" 검색
    한국어 Git/GitHub 튜토리얼 영상이 많아요. 영상으로 배우는 걸 선호하면.

Markdown과 YAML

  • Markdown Guide
    마크다운 문법의 기초부터 고급까지. 깔끔하게 정리된 레퍼런스.
  • CommonMark 튜토리얼
    10분 만에 마크다운 기초를 배울 수 있는 인터랙티브 튜토리얼.
  • YAML 공식 사이트
    YAML 형식의 공식 스펙. 프론트매터 작성할 때 참고.
  • Learn YAML in Y Minutes
    YAML 문법을 빠르게 훑어볼 수 있는 한 페이지짜리 가이드.

오픈소스 기여 가이드

  • Open Source Guides (한국어)
    GitHub에서 만든 오픈소스 기여 가이드. 한국어 번역 제공. 필독!
  • First Timers Only
    처음 오픈소스에 기여하는 사람을 위한 자료 모음.
  • Good First Issue
    초보자가 기여하기 좋은 이슈들을 모아둔 사이트. 첫 PR을 보낼 프로젝트를 찾을 때.
  • Hacktoberfest
    매년 10월에 열리는 오픈소스 기여 이벤트. 처음 시작하기 좋은 계기.

한국 개발자 커뮤니티

  • GitHub Trending (한국어)
    한국어로 된 인기 GitHub 프로젝트를 볼 수 있어요.
  • Velog
    한국 개발자들이 가장 많이 사용하는 기술 블로그 플랫폼. AI 관련 글도 많아요.
  • Disquiet
    IT 사이드 프로젝트와 스타트업 커뮤니티. AI 도구와 스킬 관련 이야기도 활발.
  • Meetup (AI 관련)
    AI 관련 오프라인/온라인 밋업을 찾을 수 있어요.
  • 카카오 오픈채팅 / 디스코드
    "Claude Code", "AI 개발", "프롬프트 엔지니어링" 등으로 검색하면 활발한 채팅방을 찾을 수 있어요.

AI 학습 자료

  • DeepLearning.AI
    Andrew Ng 교수의 AI 학습 플랫폼. 무료 강좌부터 시작 가능. "AI for Everyone" 코스 추천.
  • Coursera AI 강좌 (한국어)
    한국어로 된 AI 관련 온라인 강좌들.
  • Google ML Crash Course (한국어)
    Google이 제공하는 무료 머신러닝 입문 강좌. 한국어 지원.

유용한 도구들

  • Visual Studio Code (VS Code)
    무료 코드 에디터. SKILL.md 작성할 때 마크다운 미리보기 기능이 편리해요.
  • GitHub Desktop
    Git을 GUI(그래픽)로 사용할 수 있는 프로그램. 터미널이 어려우면 이것부터 시작.
  • Dillinger
    온라인 마크다운 에디터. 왼쪽에 마크다운, 오른쪽에 미리보기. 설치 없이 바로 사용.
  • YAML Lint
    YAML 문법이 맞는지 온라인으로 확인. 프론트매터 에러가 의심될 때 유용.
